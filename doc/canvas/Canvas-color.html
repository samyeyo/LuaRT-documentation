Canvas.color read/write property


Get or set the Canvas drawing color.

  • The color is represented by a number, an RGBA value (one byte per primary color and one byte for alpha channel).
  • A RGBA color can be represented as an hexadecimal number : 0xRRGGBBAA , RR meaning a 8bit hexadecimal red value, and so on.

Example

local ui = require "ui" require "canvas" -- create a simple window local win = ui.Window("canvas.color sample", "fixed", 500, 400) local canvas = ui.Canvas(win) canvas.align = "all" canvas:clear(0x000000FF) canvas:flip() local blue = 0x0000FFFF --Pure Blue local red = 0xFF000088 --Red with some transparency local magenta = 0xFF00FFFF -- Magenta canvas.sync = false win:show() -- update user interface repeat canvas:begin() canvas:fillcircle(300, 120, 50, blue) canvas:fillcircle(60, 300, 56, magenta) canvas:fillcircle(40, 20, 300, red) canvas:flip() ui.update() until not win.visible
