Progressbar:onDrop(kind, content) event


This event is fired when the user has dropped the mouse over the Progressbar.

Parameters

kind

A string representing the kind of content dropped on the Progressbar.

content

The content dropped on the Progressbar, the type of which depends on the following table:

kind content
"text" string
"files" table
(list of File and Directory)
"unkown" nil

Return value

The event returns no value.

Example

local ui = require "ui" -- create a window local win = ui.Window("Progressbar:onDrop() sample", 640, 480) local widget = ui.Progressbar(win) widget.allowdrop = true widget:center() -- set a onDrop() event function widget:onDrop(kind, content) ui.info("kind: "..kind.."\ncontent: "..tostring(content), "Content dropped on Progressbar") end ui.run(win):wait()
